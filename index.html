About me

Code has become my playground – a place where I love tinkering and building innovative solutions.

Sure, I might not have the fancy computer science degree, but that didn't stop me from learning programming.

My first project in C? Simple Shell – messy code and all! But it opened my eyes to the low-level workings of things. It taught me a ton about algorithm and data structure complexities.

Wanting to push myself further, I tackled building a Monty interpreter in C. Parsing out the language and making it all work was a wild ride, but it opened the door to the world of web development. That's where I built an Airbnb clone using Python, JavaScript, HTML, CSS, and MySQL. It was a beast of a project, forcing me to learn the whole stack – backend logic, user interface design, and how to manage databases.

One of my proudest creations is a school management web app. It uses a whole toolbox of technologies: Python, JavaScript, HTML, CSS, MySQL, APIs, AJAX, and Jinja templating.

This project wasn't just about all the tech I learned, but also about making sure it was easy to use and the data was secure.

Beyond those projects, I've become comfortable with languages like C, Python, and JavaScript, and can navigate databases like MySQL and NoSQL. AJAX for dynamic web interactions, Jinja templating for streamlining web development, and Redis for caching and data structures – I've got those in my toolbelt too.

let's not forget Bash scripting, network basics, and API integrations – all rounding out my skills for tackling any software challenge.

New problems are my jam, and every project is a chance to learn. If you're looking for someone who not only codes well, but who's passionate about building user-friendly and reliable software, let's chat!
